## OBS Deployment Quick Reference

Each scene manifest contains a full OBS scene configuration. General setup:
1. Add each `.mp4` as a **Media Source** — enable Loop, Use Hardware Decoding
2. Layer scene-specific ASMR audio at **10% volume** on a dedicated audio track
3. Load the Lyria 3 playlist (30 tracks per scene) on the main music track
4. Use **OBS Advanced Scene Switcher** plugin to rotate scenes on a timer (recommended: 20–40 min per scene for 24/7 rotation)
5. Optional: Add a transparent browser source overlay at 1920×1080 for channel branding/lower thirds
